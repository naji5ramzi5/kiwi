# Fresh - Customer Mobile App (Flutter)

هذا المجلد مخصص لتطبيق العميل (Customer App).
سيتم بناء هذا التطبيق باستخدام **Flutter** ليدعم أجهزة الـ iOS والـ Android.

## التقنيات المستخدمة:
- Flutter / Dart
- Supabase (للربط بقاعدة البيانات)
- Firebase Cloud Messaging (FCM) للإشعارات

> التطبيق سيعتمد نفس الهوية البصرية (اللون الأخضر والأبيض).
